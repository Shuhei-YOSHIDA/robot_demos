## RBDyn
[RBDyn](https://github.com/jrl-umi3218/RBDyn)

Install for ubuntu 14.04 and 16.04
```console
sudo add-apt-repository ppa:pierre-gergondet+ppa/multi-contact-unstable
sudo apt-get update
sudo apt-get install librbdyn-dev librbdyn-doc
```
